# Interactive

Safer interactive shell defaults for file operations.
Adds confirmation prompts when running destructive commands.

## Functions
- None.

## Aliases
- `cp`: Run `cp -i` in interactive shells.
- `ln`: Run `ln -i` in interactive shells.
- `mv`: Run `mv -i` in interactive shells.
- `rm`: Run `rm -I` in interactive shells.
- `rma`: Run `rm -rfI` in interactive shells.
